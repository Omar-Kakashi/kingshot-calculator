## üî® Forgehammer Calculator

### Purpose
Calculate the forgehammers and mythic gear needed to upgrade your mastery levels.

### How to Use

1. **Enter Current Mastery Level** (1-20)
   - Your current mastery level

2. **Enter Target Mastery Level** (1-20)
   - The level you want to reach
   - Must be higher than current level

3. **Enter Monthly Hammer Income**
   - How many forgehammers you earn per month
   - Typical range: 150-300 from Bear Trap events

4. **Click "Calculate"**

### Understanding Results

#### Total Forgehammers Needed
The total number of forgehammers required to reach your target level.

#### Timeline to Completion
Estimated time based on your monthly income:
- Shows months or years
- Helps with goal planning

#### Mythic Gear Needed
Number of mythic gear pieces to sacrifice:
- Only required for levels 11-20
- Shows total gear required

#### Level Breakdown Table
Detailed information for each level:
- Hammers required per level
- Mythic gear per level (if applicable)
- Cumulative total

### Tips & Strategies

**Levels 1-10:**
- Most cost-effective progression
- No gear sacrifice required
- Aim to reach level 10 first

**Levels 11-20:**
- Requires mythic gear sacrifice
- Significantly more expensive
- Plan carefully before upgrading

**Monthly Income:**
- Participate in Bear Trap events daily
- Join alliance activities
- Save hammers for planned upgrades

## Exporting Results

### Individual Calculator Export

Each calculator has its own "Export Results" button:

1. Complete a calculation
2. Click "Export Results"
3. A text file will download
4. Contains all details for that calculator

### Export All (Summary Tab)

For a complete report:

1. Go to Summary tab
2. Click "Export All Results"
3. Downloads comprehensive report
4. Includes all three calculators

### File Format

Exported files are plain text (.txt):
- Easy to read
- Can be edited
- Shareable with alliance
- Good for record keeping

## Troubleshooting

### Calculation Errors

**"Target level must be greater than current level"**
- Ensure target is higher than current
- Check your input values

**"Please enter a valid number"**
- Only enter numeric values
- No decimals for levels
- Check for extra spaces

### Data Not Saving

- Enable cookies/LocalStorage
- Don't use private browsing for persistent data
- Check browser settings

### Export Not Working

- Allow downloads in browser
- Check download folder
- Disable pop-up blockers
