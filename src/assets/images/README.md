# Kings Shot Calculator - Game Images

This directory contains images from the Kings Shot mobile game used in the calculator interface.

## Image Guidelines

### Required Images:

1. **Hero Images** (`heroes/`)
   - Individual hero portraits (Amadeus, Helga, Chenko, etc.)
   - Recommended size: 256x256px
   - Format: PNG with transparency

2. **Pet Images** (`pets/`)
   - Wolf, Lynx, Bison, Cheetah, Moose, Lion, Bear
   - Golden Mighty Bison, Giant Rhino
   - Recommended size: 256x256px
   - Format: PNG with transparency

3. **Gear Images** (`gear/`)
   - Coat, Pants, Ring, Weapon, Hat, Necklace
   - Tier icons (Green, Blue, Purple, Gold, Red)
   - Recommended size: 128x128px
   - Format: PNG with transparency

4. **Resource Icons** (`resources/`)
   - Forgehammer icon
   - Satin icon
   - Gilded Threads icon
   - Artisan's Vision icon
   - Charm Guides icon
   - Charm Designs icon
   - Pet Food icon
   - Recommended size: 64x64px
   - Format: PNG with transparency

5. **Game Screenshots** (`screenshots/`)
   - Bear Pitfall event
   - Charm upgrade screen
   - Governor gear enhancement
   - Pet leveling interface
   - Recommended size: 1920x1080px or original resolution
   - Format: JPG or PNG

## File Naming Convention:

- Use lowercase with hyphens
- Examples:
  - `heroes/amadeus.png`
  - `pets/golden-mighty-bison.png`
  - `gear/red-t4-weapon.png`
  - `resources/forgehammer.png`
  - `screenshots/bear-pitfall-event.jpg`

## Adding Images:

To add images to the calculator:

1. Place images in appropriate subdirectories
2. Update the corresponding HTML/CSS to reference the images
3. Optimize images for web (compress, appropriate size)
4. Ensure transparent backgrounds where applicable

## Image Optimization:

Recommended tools:
- TinyPNG (https://tinypng.com/) - PNG compression
- Squoosh (https://squoosh.app/) - Multi-format compression
- ImageOptim (Mac) - Batch optimization

Target file sizes:
- Icons: < 10KB each
- Hero/Pet portraits: < 50KB each
- Screenshots: < 300KB each
